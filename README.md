# 🌱 Agro Forte, Futuro Sustentável

Projeto de Front-End desenvolvido com o tema:

**“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.”**

O objetivo do projeto é apresentar, de forma visual e informativa, a importância da sustentabilidade no agronegócio, mostrando como a produção agrícola pode crescer sem comprometer os recursos naturais.

---

# 📌 Objetivo do Projeto

Este projeto busca:

- Demonstrar a relação entre tecnologia, agricultura e sustentabilidade;
- Incentivar a conscientização ambiental;
- Apresentar um layout moderno e responsivo utilizando HTML, CSS e JavaScript;
- Desenvolver habilidades de criação de interfaces front-end.

---

# 🖥️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
